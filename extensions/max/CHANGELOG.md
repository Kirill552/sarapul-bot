# Changelog

All notable changes to the MAX Messenger channel plugin will be documented in this file.

## [2026.2.12] - 2026-02-12

### Added

- Initial release of MAX Messenger channel plugin
- Support for text messages with Markdown/HTML formatting
- Media support (images, videos, audio, files)
- Inline keyboard buttons
- Message editing and deletion
- Callback button handling
- Bot start events (`bot_started`)
- Bot added/removed events (`bot_added`, `bot_removed`)
- CLI onboarding wizard
- Multi-account support
- Health check and status probing
